# JCSport_IOS App
IOS 스포츠 소셜앱 입니다. 원래 계속 진행하려 했으나 업무나 여러 개인상황들로 인해 현재는 잠정중단한 상태입니다. 로그인 기능까지 구현했고 , 추후에 개인 시간적 여유가 있다면 개발에 들어가려 생각중입니다.
파일 업로드 기능 , 소셜 로그인 기능 등을 적용했습니다.
